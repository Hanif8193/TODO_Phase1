# Project & Time Management

## Description
Plan, track, and coordinate tasks for hackathon MVP.

## Key Tasks
- Break down work into sprints or subtasks
- Prioritize core features over extras
- Monitor progress and deadlines
- Prepare demo flows for judges

## Associated Agent
**Project Manager / Hackathon Strategist**

## Scope
- Sprint planning and task breakdown
- Feature prioritization for MVP delivery
- Progress tracking and status reporting
- Risk identification and mitigation
- Demo preparation and presentation planning
- Timeline coordination across agents

## Success Criteria
- [ ] Work is broken into manageable, trackable tasks
- [ ] MVP features are clearly prioritized
- [ ] Progress is visible and blockers are identified early
- [ ] Demo script is prepared and rehearsed
- [ ] Deadline is met with core functionality complete
